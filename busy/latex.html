3. Convergence of Adaptive SGD

3.1 Problem Setting

We consider the non-convex stochastic optimization problem

minx∈ℝd   f(x) = 𝔼ξ∼𝒟 [F(x; ξ)], (3.1)

where f: ℝd → ℝ is possibly non-convex, L-smooth, and bounded below by f★ > -∞. The stochastic oracle returns unbiased gradients g(x; ξ) satisfying 𝔼[g(x;ξ)] = ∇f(x).

Assumption 3.1 (Bounded variance). For all x ∈ ℝd, there exists σ > 0 such that 𝔼[‖g(x;ξ) − ∇f(x)‖²] ≤ σ².

3.2 Main Result

Theorem 3.2 (Convergence rate of AdaNova). Let Assumptions 3.1–3.2 hold. Fix ηt = η / √t for t ≥ 1, β1 ∈ [0,1), and β2 ∈ [β1, 1). Then the iterates of Algorithm 1 satisfy

mint≤T 𝔼‖∇f(xt)‖² ≤ 𝒪( f(x0)−f★ / η√T  +  Lησ² log T / √T ) (3.2)

In particular, choosing η = Θ(1/√L) yields the optimal 𝒪(T−1/2) convergence up to logarithmic factors.

The proof, deferred to Appendix A, follows the descent-lemma template of Ghadimi and Lan (2013) combined with a novel variance-coupled step-size bound that exploits Assumption 3.1 to control the second-moment estimator.
